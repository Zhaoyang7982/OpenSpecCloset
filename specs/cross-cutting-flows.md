# 核心流程链路（端到端）

## FLOW-1 冷启动与模式选择

1. **未启用云端** → 直接进入衣橱（本机数据）。  
2. **启用云端 + 仅本机** → 直接进入衣橱（本机数据，与账号隔离）。  
3. **启用云端 + 非仅本机 + 未登录** → 登录/注册。  
4. **已登录** → 使用云端衣橱主路径。

## FLOW-2 单品录入闭环

选图 →（可选）处理图 → 填信息 → 保存 → 衣橱列表可检索 → 详情可编辑。

## FLOW-3 搭配创建与使用闭环

创建搭配 → 保存 → 搭配列表可筛选查看 → 详情可分享 → 可移入回收站/恢复/彻底删除。

## FLOW-4 日历规划与记录闭环

在日历为某日关联计划穿搭 / 标记已穿 → 月视图可见标记 → 可写备忘与设提醒（支持平台）→ 统计页可消费穿着记录。

## FLOW-5 今日推荐闭环

拉取今日语境（按端能力降级）→ 读取衣橱与历史穿着 → 生成至多四套 → 用户查看详情 → 下拉刷新可重复触发（受智能额度与端策略约束）。

## FLOW-6 同步闭环（启用云端且非仅本机）

本地修改 → 若离线则进入待同步队列并计数 → 网络恢复或用户手动同步 → 队列重放 → 列表与详情与远端一致化（冲突处理以实现为准；需求层要求最终可感知的一致性，不要求在本规格展开具体合并算法）。
